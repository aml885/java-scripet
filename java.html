1. How to select elements in the DOM?

আইডি ব্যবহার করে নির্বাচন করা → নির্দিষ্ট আইডি অনুযায়ী এলিমেন্ট নির্বাচন করা যায়। ক্লাস ব্যবহার করে নির্বাচন করা → নির্দিষ্ট ক্লাস অনুযায়ী একাধিক এলিমেন্ট নির্বাচন করা যায়। ট্যাগ ব্যবহার করে নির্বাচন করা → নির্দিষ্ট ট্যাগ অনুযায়ী এলিমেন্ট খোঁজা যায়। CSS সিলেক্টর ব্যবহার করে নির্বাচন করা → নির্দিষ্ট সিলেক্টরের মাধ্যমে প্রথম মিল থাকা এলিমেন্ট রিটার্ন করা হয়। CSS সিলেক্টর ব্যবহার করে সব মিল থাকা এলিমেন্ট নির্বাচন করা → নির্দিষ্ট সিলেক্টরের সব মিল থাকা এলিমেন্টের তালিকা রিটার্ন করা হয়।

2. Difference between innerHTML, innerText, and textContent?

এটি HTML ট্যাগসহ সম্পূর্ণ কনটেন্ট নেয় বা পরিবর্তন করে। এটি শুধুমাত্র দৃশ্যমান লেখা নেয় বা পরিবর্তন করে, লুকানো লেখা উপেক্ষা করে। এটি HTML ট্যাগ ছাড়া সম্পূর্ণ লেখা নেয়, এমনকি লুকানো থাকলেও।

3. What is event delegation in the DOM?

ইভেন্ট ডেলিগেশন হলো একটি কৌশল যেখানে শিশু এলিমেন্টগুলোর জন্য আলাদা ইভেন্ট যোগ না করে প্যারেন্ট এলিমেন্টে ইভেন্ট যোগ করা হয়। এটি ইভেন্ট বাবলিং ব্যবহার করে, যেখানে ইভেন্ট নিচ থেকে ওপরের দিকে ছড়িয়ে পড়ে। এটি পারফরম্যান্স বাড়ায় এবং কম এলিমেন্টে ইভেন্ট হ্যান্ডলিং করতে সাহায্য করে।

4. What is event bubbling in the DOM?

ইভেন্ট বাবলিং হলো একটি প্রসেস যেখানে কোনো ইভেন্ট প্রথমে টার্গেট এলিমেন্টে ঘটে এবং তারপর তার প্যারেন্ট এলিমেন্টের দিকে উপরের দিকে ছড়িয়ে পড়ে। এটি ডিফল্টভাবে কাজ করে এবং প্যারেন্ট এলিমেন্টে ইভেন্ট শোনার সুবিধা দেয়। ইভেন্ট বাবলিং ব্যবহার করে একটি প্যারেন্ট এলিমেন্টের মাধ্যমে অনেক চাইল্ড এলিমেন্টের ইভেন্ট হ্যান্ডেল করা যায়।

5. How do you create, add, and remove elements using JavaScript?

জাভাস্ক্রিপ্ট দিয়ে নতুন HTML উপাদান তৈরি করা যায় document.createElement() ফাংশন ব্যবহার করে। তৈরি করা উপাদানটি DOM-এ যোগ করতে appendChild() বা append() ফাংশন ব্যবহার করা হয়। একটি উপাদান মুছে ফেলার জন্য remove() বা removeChild() ব্যবহার করা হয়।
